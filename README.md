# Ejercicio de Feedback - Unidad Didáctica 2

**Alumno:** Andrés González Álvarez  
**Asignatura:** Informática 2 - UAX  

## Tecnologías utilizadas
- HTML5 semántico
- CSS3 (Flexbox y Grid)
- Diseño responsive con Media Queries

## Estructura del proyecto
- `index.html`: listado de artículos (desktop y mobile).
- `article.html`: detalle de artículo (desktop y mobile).
- `readme.html`: página informativa (desktop y mobile si aplica).
- `css/style.css`: estilos globales y media queries.
- `asssets/img/`: recursos de imagen exportados del diseño.

## Ejecución
Abrir `index.html` en el navegador. Desde ahí se puede navegar al resto de páginas mediante enlaces.
